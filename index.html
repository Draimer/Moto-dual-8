Core Experience

不是普通賽車，是桌面上的雙人決鬥。

MotoDuel 把滑鼠變成摩托車把手。玩家不只是在螢幕上比速度， 還會從手上的滑鼠感覺到壓車、撞擊和加速節奏。

P1/P2

一台電腦兩人 PK

兩顆滑鼠同時接上同一台 Windows 電腦，遊戲會分別辨識 P1 / P2，坐在同一台螢幕前就能開戰。

HAPTIC

滑鼠傾斜與震動

滑鼠不只是指標。車身壓彎、擦撞、加速出彎時，回饋會同步傳回玩家手上。

DUEL

雙車貼身競爭

左右分割畫面同步衝刺，搶線、卡位、射擊，最後一圈才是真正的壓力測試。

TRACK

四條經典賽道

Monza、Sachsenring、Sochi、Yas Marina。高速直線、技術彎、夜景賽道一次滿足。

AI

五級 AI 對手

從練習到殘酷難度，一個人也能熱身；AI 會走線、閃障礙，也會製造壓迫感。

SHOT

射擊與心理戰

射擊會讓對手短暫減速，喇叭則是純粹干擾。速度、路線和膽量都會影響勝負。

Live Demo

兩顆滑鼠，就是兩台摩托車的把手。

Demo 的重點是「同一台電腦，兩個人，各拿一顆滑鼠」。滑鼠左右移動控制車身傾斜， 撞擊和加速會觸發震動，讓玩家真的感覺自己在跟對手互相壓車。

  • 遊戲獨立辨識兩顆 USB 滑鼠，P1 / P2 不會互相干擾。
  • 滑鼠移動對應摩托車壓車角度，畫面中的車身同步傾斜。
  • 碰撞、射擊命中和加速節奏會產生震動回饋。
Controls

一邊按鍵加速，一邊用滑鼠控車。

P1 用 WASD，P2 用方向鍵。滑鼠控制轉向與車身手感，鍵盤負責油門、煞車和換檔。

Requirements

Windows 電腦接上兩顆滑鼠就能玩。

遊戲以 Electron + WebGL 2 製作，雙人模式建議準備兩顆 USB 滑鼠。
